# Desafio Técnico - Processo Seletivo INJunior

## Sobre o Projeto

Este repositório contém a resolução dos exercícios propostos no desafio técnico do processo seletivo da INJunior.

Os exercícios foram desenvolvidos em JavaScript com foco na aplicação de conceitos fundamentais de lógica de programação, estruturas de controle, funções, arrays e objetos.

## Como Executar

### Exercícios com HTML

1. Abra a pasta do exercício desejado.
2. Execute o arquivo `.html` em qualquer navegador.
3. Siga as instruções exibidas por meio de caixas de diálogo (`prompt` e `alert`).

## Objetivos Trabalhados

Durante o desenvolvimento dos exercícios foram praticados conceitos como:

* Variáveis e tipos de dados;
* Estruturas condicionais (`if`, `else`, `switch`);
* Estruturas de repetição (`for`, `while`, `do...while`);
* Funções;
* Arrays;
* Objetos;
* Manipulação de strings;
* Lógica de programação.
